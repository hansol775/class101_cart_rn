# class101_cart_rn
클래스101 프론트엔드 개발자 과제 - React Native

## install

```
git clone git@github.com:hansol775/class101_cart_rn.git

cd class101_cart_rn

npm install

npx pod-install

react-native run-ios & run-android
```
